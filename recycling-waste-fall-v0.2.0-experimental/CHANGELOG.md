# v0.2.0-experimental

## Experimental Falling Waste Game

Initial experimental release of the falling-waste variant.

Features:

- Falling waste well
- Multiple waste types
- Different waste shapes
- Different falling speeds
- Click-to-match recycling mechanic
- Pair removal system
- Score counter
- Timer
- Level progression
- Restart button
- High-score saving with LocalStorage
- Static hosting compatibility

Notes:

- This is an experimental branch.
- The stable card game remains unchanged.
- No backend required.
- No frameworks required.
