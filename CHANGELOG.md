# Changelog

## v0.2.0-experimental

**Experimental Falling Waste Game**

Initial experimental release of the falling-waste variant
(`recycling-waste-fall-v0.2.0-experimental/`). The stable card game remains
unchanged.

### Features

- Falling waste well
- Multiple waste types
- Different waste shapes
- Different falling speeds
- Click-to-match recycling mechanic
- Pair removal system
- Score counter
- Timer
- Level progression
- Restart button
- High-score saving with LocalStorage
- Static hosting compatibility

### Notes

- This is an experimental branch.
- The stable card game remains unchanged.
- No backend required.
- No frameworks required.

## v0.1.0-stable-core

**Initial Stable Release**

### Features

- 48-card deck
- 12 waste categories
- Recycling bin mechanics
- Pair matching system
- Bonus scoring system
- Dangerous waste penalties
- Doom timer
- High-score table
- LocalStorage support
- Shared hosting compatibility

### Technical Notes

- HTML5
- CSS3
- Vanilla JavaScript
- No backend
- No framework
- GitHub-ready
